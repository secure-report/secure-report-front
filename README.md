# Secure Report - App de Denuncias Anónimas

## Descripción

Secure Report es una aplicación móvil centrada en la protección del ciudadano que permite denunciar irregularidades en productos y servicios de manera completamente anónima. La app ofrece formularios guiados, carga de fotos y videos, ubicación automática y selección de categorías como precios abusivos, mala calidad o mal servicio. Se enfoca en generar confianza mediante garantías de anonimato, utilizando cifrado avanzado y evitando el rastreo de datos personales. Ideal para reportar malas prácticas en el servicio al cliente y promover un entorno más justo y transparente.

## Características Principales

- **Anonimato Total**: Denuncias sin revelar identidad, con cifrado de datos y no rastreo de información personal.
- **Formularios Guiados**: Interfaz intuitiva que guía al usuario a través del proceso de denuncia.
- **Carga de Multimedia**: Posibilidad de adjuntar fotos y videos como evidencia.
- **Ubicación Automática**: Detección automática de la ubicación para contextualizar las denuncias.
- **Categorización**: Selección de categorías específicas como precios abusivos, mala calidad o mal servicio.
- **Confianza y Seguridad**: Enfoque en la protección de datos y privacidad del usuario.

## Tecnologías Utilizadas

El frontend de la aplicación está desarrollado con las siguientes tecnologías:

- **React Native**: Framework para el desarrollo de aplicaciones móviles nativas usando JavaScript y React.
- **Expo**: Plataforma para el desarrollo rápido de apps React Native, facilitando la construcción y despliegue.
- **TypeScript**: Lenguaje de programación que añade tipado estático a JavaScript para mayor robustez y mantenibilidad.
- **NativeWind**: Adaptación de Tailwind CSS para React Native, permitiendo estilos utilitarios y consistentes.
- **Metro**: Bundler de JavaScript para React Native, optimizado para el desarrollo móvil.
- **ESLint y Prettier**: Herramientas para linting y formateo de código, asegurando calidad y consistencia.
- **Babel**: Transpilador de JavaScript para compatibilidad y optimización.

## Instalación

Para instalar y ejecutar la aplicación en tu entorno local, sigue estos pasos:

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/tu-usuario/secure-report-front.git
   cd secure-report-front/frontend/my-expo-app
   ```

2. **Instala las dependencias**:
   ```bash
   npm install
   ```

3. **Inicia la aplicación**:
   ```bash
   npx expo start
   ```

4. **Ejecuta en un dispositivo o emulador**:
   - Para Android: Presiona `a` en la terminal o escanea el código QR con la app Expo Go.
   - Para iOS: Presiona `i` o escanea el código QR con la app Expo Go en un dispositivo iOS.

## Uso

Una vez instalada, la app permite a los usuarios crear denuncias anónimas siguiendo estos pasos:

1. Abre la app y selecciona la opción de nueva denuncia.
2. Completa el formulario guiado, seleccionando la categoría relevante.
3. Adjunta fotos o videos si es necesario.
4. La ubicación se detectará automáticamente.
5. Envía la denuncia de forma segura y anónima.
